# aner admin 5

基于 `Dcat admin` 开发的通用后台管理系统。内置了会员模块、会员资金模块、系统设置模块、广告设置模块、文章模块、公告模块、日志记录模块等。
并且内置了登录、会员信息、文章、支付等项目通用API。


#### 安装流程
进入项目根目录下，输入一下代码：
```
composer install
或
composer update
```
如果出现问题，则需要开启`php`的 `fileinfo` 扩展.
